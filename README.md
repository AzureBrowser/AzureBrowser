## 👋 Hi there

Welcome to the official GitHub profile for **Azure Browser**.

Azure Browser is a modern browser project focused on delivering a **lightweight**, **clean**, and **desktop-quality** browsing experience.  
It is built for users who want a browser that feels fast, polished, and simple to use — without unnecessary clutter.

## 🧩 Built In

Azure Browser is planned around useful built-in essentials, including:

- 📜 **Local history**
- ⬇️ **Download manager**
- 🔑 **Password manager**
- 🔄 **Import tools**
- 🛡️ **Site safety tools**
- 💾 **Local-first data storage**
- 🌙 **Dark theme**
- 📱 **PWA install support**
- 🖥️ **Desktop app window experience**
- ⚙️ **Customizable interface sizing**

## 🧠 Why Azure Browser?

Azure Browser is built around a simple idea:

> **Make a browser that looks modern, feels smooth, and stays easy to use.**

Instead of feeling like just another wrapper or clone, Azure Browser aims to deliver a product that feels polished, branded, and user-friendly.

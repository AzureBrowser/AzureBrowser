# Azure Browser

## Hi there 👋

Welcome to the official GitHub profile for **Azure Browser** — a modern, lightweight desktop browser focused on speed, clean design, customization, and a better everyday browsing experience.

Azure Browser is built to feel like a real desktop application, with a polished interface, responsive window behavior, dark theme support, and user-friendly settings.

---

## 🚀 About Azure Browser

**Azure Browser** is designed for users who want a browser that feels clean, fast, and simple without unnecessary clutter.

Our goal is to create a browser experience that is:

- ⚡ Lightweight and responsive
- 🎨 Clean, modern, and customizable
- 🌙 Dark theme ready
- 🖥️ Desktop-app styled
- 📱 PWA-friendly for mobile and tablet users
- 🔧 Easy to improve, build, and expand

---

## 🧩 Current Focus

We are currently working on:

- Custom desktop browser UI
- Improved window controls
- Dark mode support
- Adjustable UI scaling in settings
- PWA/mobile install support
- Website and installer download flow
- Better resize and snap behavior on Windows
- More polished browser branding and visuals
